Тестовая работа Антоневича Георгия Сергеевича для Infotecs


Краткое руководство по работе с Таблицей


  1. Сортировка таблицы предусмотрена
    Для того чтобы отсортировать таблицу, нажмите на любой нужный столбец 1 раз
    Для сортировки в обратную сторону нажмите на тот же столбец два раза
  2. Предусмотрено редактирование любой ячейки Таблицы
    Для того чтобы отредактировать таблицу нажмите на любую ячейку Дважды!(шапка не редактируется)
    Чтобы выйти из редактирования, нажмите внутрь Div контейнера, чтобы появилась
    печатная строка, потом нажмите на любое поле на странице Html 1 раз.
  3. Для того чтобы скрыть/показать столбец, в таблице предусмотрены кнопки
    Каждая кнопка названа по номеру столбца.
    Нажмите один раз на кнопку нужного столбца чтобы его скрыть
     Нажмите на кнопку скрытого столбца Дважды!Чтобы столбец снова появился
